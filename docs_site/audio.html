Routing

Audio flows through an analyser for metering, then splits to recording and monitoring nodes.

  • Analyser: FFT/levels mapped to parameters (e.g., amount, rgbShift, melt).
  • Recorder: mixed bus for capture alongside the canvas stream.
  • Monitor: speakers for live feedback.

Some browsers require a click to resume the AudioContext. Tap anywhere if meters look frozen.
