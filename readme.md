# ERRORHEAD - Local Development

A browser-based glitch art studio using Canvas2D + WebGL. Tailwind CSS is bundled locally via PostCSS, and all dependencies are bundled via Vite.

## Run Locally

Prerequisites: Node.js (LTS)

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`

Build for production:

`npm run build` then `npm run preview` to serve the production build.

Notes:
- Tailwind is compiled locally; no CDN is used.
- No AI or external API keys are required.
