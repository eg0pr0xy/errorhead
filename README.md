Minimal browser-based glitch studio.

- Live: https://eg0pr0xy.github.io/errorhead/

Quickstart
- Run local
  - `npm i`
  - `npm run dev`
- Build
  - `npm run build`

Usage
- Open the live URL
- Drop media → bend entropy → export

Signals
- No cloud, no trackers, no keys
- Only `README.md` is public; internal docs stay dark
