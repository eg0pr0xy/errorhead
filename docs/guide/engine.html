Frame Pipeline

Frames pass through stateless glitch ops, optional temporal mosh, and WebGL shader passes.

  • Stateless: JPEG‑style corruption, RGB shifts, pixelation, scanlines.
  • Temporal: motion vectors, smear/blend, frame skipping/stepping.
  • WebGL: shader‑based distortions and post‑FX.

Determinism is opt‑in; turn it on when you need reproducible artifacts.

Key Parameters

amount/quality/iterations
Primary corruption intensity & repeat passes.

pixelation/noise/scanlines

Effects Catalog (Current)

These are the effects currently implemented in the engine. All are available in the UI unless noted.

Core / Compression

  • Corruption: amount, quality, iterations, seed
  • Corrupt Mode: safe / unsafe byte corruption

Destruction

  • Melt
  • Shred
  • Scatter
  • Burn (heat tint / blend influence)
  • Shake (frame jitter)

Color & RGB

  • Brightness
  • RGB Shift
  • Color Offset
  • Channel Offsets: red / green / blue shift
  • Pixelation

Display / Texture

  • CRT Scanlines + intensity
  • Noise (digital slivers)

Analog Sync Distortion

  • Phase Slip: phaseSpeed, phaseOffset, phaseJitter, wrapMode, banding
  • Analog Inertia: analogInertia (Verlet-style damping)
  • Horizontal Drift: hOffset, hSpeed, hAmount
  • Wave Distortion: waveAmount, waveFrequency, waveSpeed, wavePhase
  • Vertical Sync Collapse: bandCount, baseSpeed, bandVariance, jitter, wrapMode
  • VHS Tracking Band: strength, bandHeight, speed, position, tear

Additional Analog Layers

  • Chroma Drift / Delay: chromaDelayFrames, chromaDriftAmount, chromaWobbleSpeed
  • Slice / Stripe Displacement: sliceCount, thickness, offset, direction, hold
  • No-Signal Snow Burst: duration, chance, intensity

Professional Glitch

  • Pixel Sorting: brightness / hue / random
  • Channel Separation

Temporal

  • Temporal Echo
  • Time Smear
  • Reverse Burst
  • Stutter Lock
  • Temporal Displace
  • Partial Replace (patterns + offset)
  • Chaos (seed evolution)

2D Datamosh (Canvas)

  • I-Frame / GOP: interval, bloom, soft reset
  • Motion Vector Field: flowX/flowY/rotate/zoom
  • Feedback
  • Macroblocks: blockList, blockSize
  • Ref Swap + Motion Residue
  • MV Abuse (block match)
  • History Capture Interval

WebGL Mosh (Shader)

  • Motion Sculpture WebGL (when moshMode = webgl)
  • Shader Mosh: feedback, decay, warp, block size, RGB offset
  • Displacement: strength, scale, speed, quantize

Audio Reactive

  • Audio Modulation: RGB shift, amount, warp, feedback, blocks, melt, shake, noise

Strict Effects Layer (Optional)

  • blend/smear: streaks and trails via accumulative mixing.
  • frameOps: drops, holds, and reverse micro‑loops.
