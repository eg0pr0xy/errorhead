What Is ERRORHEAD?

ERRORHEAD is a compact glitch studio powered by Canvas2D + WebGL. It bends frames, scrapes motion, and sculpts compression into artifacts you can steer.

Inputs
Images, videos, microphone/music for audio-reactive flow.
Engine
Stateless glitch ops + temporal mosh + WebGL shader passes.
Outputs
Still frames (PNG/JPEG), recordings, or PNG sequences.

Core Functions

  • Import: drop an image/video or pick via the left panel.
  • Glitch Parameters: amount, quality, iterations, RGB offsets, pixelation, noise, melt/shred/scatter, brightness, scanlines.
  • Temporal: master speed, frame stepping, smear/blend, motion‑based artifacts.
  • Audio: mic, video, or music input to modulate parameters.
  • Animate: toggle animation mode; scrub timeline; set keyframes.
  • Export: still (PNG/JPEG), recording (webm/mp4 ‘auto’), or PNG sequence.

Tip: Randomize to discover new textures, then lock resolution and iterate to keep the vibe consistent.

Workflow

  • 1. Load media → check canvas dims
  • 2. Dial glitch params; toggle scanlines; set temporal speed
  • 3. Perform live (enable audio) or sketch keyframes
  • 4. Export a still, a recording, or a frame sequence

Use Cases

Glitch Stills
Album art, posters, thumbnails with deliberate artifacting.
Motion Sculpture
Temporal smear for ghost trails and compression echoes.
Live Visuals
Audio‑reactive sets with mic/music routing.
Social Clips
Quick exports with bold scanlines and RGB shift.

Signals

  • Offline‑first. No keys. No trackers.
  • All processing in‑browser; nothing leaves your machine.
  • Works on Chromium/Firefox; WebGL recommended for performance.
